#OAuth
Twitter는 OAuth를 사용하여 API에 대한 인증 된 액세스 권한을 제공한다.

##OAuth
웹, 모바일 및 데스크톱 응용 프로그램에서 간단하고 표준적인 방법으로 보안 인증을 허용하는 개방형 프로토콜이다.
